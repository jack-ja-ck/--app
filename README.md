# 敬拜投屏

一款专为教会敬拜设计的歌词管理与投屏工具，支持多设备同步显示、主领视图、动态背景等功能。

## 功能特点

- **歌词管理**：轻松创建、编辑和管理敬拜诗歌歌词
- **实时投屏**：通过浏览器窗口实现歌词投屏显示
- **主领视图**：为敬拜主领提供专用的提词界面
- **多设备同步**：支持手机、平板等设备同时访问
- **动态背景**：支持粒子动画、渐变、自定义图片等多种背景效果
- **字体定制**：可调整字体大小、类型和垂直位置
- **自动播放**：支持定时自动翻页功能
- **主题切换**：提供深色、浅色、暖色、高对比等多种主题

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击"开启投屏"按钮打开演示窗口
3. 点击"主领视图"按钮打开主领提词界面
4. 在手机浏览器中访问同一页面即可同步控制
